# Minecraft Importer
Import packages to Minecraft Bedrock Edition.

Minecraft Importer automatically imports your packages to Minecraft Bedrock Edition with only a few clicks. To be used with Minecraft Bedrock Edition Linux.

## Installation

To install:

![image](https://user-images.githubusercontent.com/72320038/116605536-88306f80-a8fd-11eb-823f-97d0b0333998.png)

## Usage

To run Minecraft Importer, click Menu > Games > Minecraft Importer.
An import guide will appear on your screen as it automatically imports your package.
Minecraft Importer can import any package, such as a world or a texture pack.

Accepted Packages:
.mcworld (Minecraft World)
.mcpack (Minecraft Resource/Behavior Pack)

Minecraft Importer was designed for Debian distros.
